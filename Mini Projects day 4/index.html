Tasklist
This is a simple web-based task list application built with HTML, CSS and JavaScript. The application allows the user to add and store tasks in the local storage of the user's browser. The tasks are displayed in a list with a delete icon next to each task. The user can delete a task by clicking the icon and the task will be removed from both the local storage and the list on the screen. The application also provides a button to clear all tasks from the local storage and the screen. This is a basic and efficient task management tool that can be easily accessed and used anywhere, without the need for an internet connection.
